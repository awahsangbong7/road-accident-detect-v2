   - Normalizes pixel values to [0,1]

3. Include progress tracking

PHASE 2 - Detection & Tracking:
1. Implement YOLODetector class:
   - Load YOLOv8n model
   - Detect vehicles, persons, motorcycles
   - Filter detections by confidence (>0.5)
2. Implement DeepSORTTracker class:
   - Track objects across frames
   - Maintain trajectory history
   - Handle object entry/exit

2. Calculate accident confidence score
3. Trigger alert only if confidence >90%

PHASE 4 - GPS & Alerts:
1. GPSManager class:
   - Register camera locations (lat/lon)
   - Assign GPS to detected accidents

   - Send via Twilio API
   - Log all alerts to database
